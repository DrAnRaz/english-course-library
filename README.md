# English Course Library

70 interactive English lessons for kids & teens (9–15), organized in 7 levels (0–6 → pre-A1 to C2), designed for one-to-one Zoom teaching with a neurodivergent-first template.

**Live site:** https://dranraz.github.io/english-course-library/

- `index.html` — master hub (level picker)
- `00_ASSESSMENT/placement.html` — 20-min guided placement test → recommends starting level
- `00_ASSESSMENT/scoring_guide.html` — teacher-only scoring anchors & rules
- `TEACHER_GUIDE.html` — Zoom workflow, teacher mode, ND design usage
- `Level_X_*/session01.html … session10.html` — the 70 sessions
- `CURRICULUM.md` — locked curriculum map & template spec

Every session: same 8-step ritual, ≈50 min core with skip-if-short markers, click-to-hear audio (browser TTS), self-checking quizzes with kind feedback, movement break, embedded challenge game, teacher mode (PIN), local auto-save + .txt export. No external resources; works offline.
