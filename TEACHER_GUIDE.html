👩‍🏫 Teacher Guide — English Course Library

Everything you need to run any of the 70 sessions over Zoom. 5-minute read.

1 · Before the first lesson with a NEW student

  • Run the Placement Assessment (~20 min) → it recommends Level 0–6.
  • Read the Scoring Guide once — it has the level anchors and decision rules.
  • Open the recommended level's session01.html and skim it in teacher mode (see §3).

    3 · Teacher mode 🔐

  • Click the red 🔐 button (bottom right) → PIN: teach
  • Reveals: per-step timings in the colored headers, 📹 Zoom cues, "Say:" goal scripts, per-step teacher notes, answer keys, self-evaluation boxes. The student's plain view shows none of this — they only see learning content.
  • Click 🔓 Teacher again to hide everything (e.g. before projecting your window anywhere).
    • ⚠️ Teacher mode belongs on YOUR screen only. In the default flow the student is on their own device, so nothing can leak. In fallback dual-window mode, only ever share the window where teacher mode is OFF.

    4 · Timing & the skip system

  • Every session: ≈50 min core. The 8 steps have honest per-step timings in the colored headers — visible in teacher mode only, so the student never feels the clock.
  • Running late? Every session has 2–3 ⏭ SKIP-IF-SHORT boxes — cut those parts guilt-free; they convert to homework or next lesson's warm-up.
  • Golden rule: never cut Step 6 (Creative) or the MOVE break — cut Step 4's written questions and Challenge Round 3 first.

    • 5 · Neurodivergent-friendly design — how to use it

  • Same ritual every session (8 steps, same order, same colors) — announce steps by number: "We're on Step 3!"
  • Agenda pills at the top: let the STUDENT tap each step done — closure and dopamine.
  • MOVE break is not optional decoration; it resets attention. Do it together.
  • 🔊 buttons everywhere: click-to-hear any word or the whole reading (computer voice). Use it twice for processing time; never rush a reply.
  • Readings are chunked with mini-headings; ask for ONE chunk at a time.
  • Quizzes give kind feedback and reveal the right answer in green — treat wrong answers as material ("why is the green one right?").
